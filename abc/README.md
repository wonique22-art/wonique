# 팀 랜덤 선택기 (Team Random Selector)

모던하고 심플한 팀 의사결정 웹앱입니다. 회식 메뉴 선택, 발표 순서 정하기 등에 사용할 수 있습니다.

## 🚀 기능

- **🎯 랜덤 선택**: 항목 중 하나를 무작위로 선택합니다
- **🔀 순서 정하기**: 모든 항목을 랜덤 순서로 배열합니다
- **📥 파일 업로드**: `.txt` 파일로 항목을 대량 입력합니다
- **🔗 URL 파라미터**: `/?items=항목1,항목2,항목3` 형식으로 빠른 접근
- **📱 반응형 디자인**: 모바일, 태블릿, 데스크톱 모두 최적화
- **✨ 매끄러운 애니메이션**: 선택 과정이 재미있고 시각적입니다

## 🛠️ 기술 스택

- **Frontend**: [Next.js 14](https://nextjs.org/)
- **CSS**: [Tailwind CSS](https://tailwindcss.com/)
- **Language**: TypeScript
- **Animation**: CSS Animations

## 📖 사용 방법

### 1. 직접 입력
- 홈페이지 텍스트 에어러어에 항목을 입력합니다
- 쉼표(`,`) 또는 줄바꿈(`\n`)으로 항목을 구분합니다
- 예: `김철수, 이영희, 박민준` 또는 각각 한 줄씩 입력

### 2. 파일 업로드
- `.txt` 파일을 준비합니다 (한 줄에 한 항목)
- 업로드 영역으로 파일을 드래그 & 드롭하거나 클릭합니다
- 자동으로 항목이 추가됩니다

### 3. URL 파라미터
- 아래와 같은 형식으로 URL을 공유합니다:
  ```
  https://yoursite.com/?items=팀원1,팀원2,팀원3
  ```

### 4. 선택 또는 순서 정하기
- **랜덤 선택**: 항목 중 하나를 무작위로 선택 (회식 메뉴 선택 등)
- **순서 정하기**: 모든 항목을 무작위 순서로 배열 (발표 순서 정하기 등)

## 🎨 디자인 특징

- **모던 & 심플**: 클린한 라이아웃과 미니멀한 디자인
- **활기찬 색감**: 그래디언트를 활용한 시각적으로 매력적인 UI
- **부드러운 애니메이션**: 선택 과정을 흥미롭게 표현
- **직관적인 UX**: 1-2 클릭으로 의사결정 완료 가능
- **모바일 최적화**: 모든 기기에서 쾌적한 사용성

## 🤝 사용 사례

1. **회식 메뉴 선택**: 팀원 목록에서 한 명을 선택하여 그 사람이 메뉴를 정하게 합니다
2. **발표 순서**: 발표자 목록의 순서를 랜덤하게 정합니다
3. **미팅 주제**: 토론할 주제를 무작위로 선택합니다
4. **팀빌딩 활동**: 게임이나 활동의 순서를 결정합니다
